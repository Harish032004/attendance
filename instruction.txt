Great! There are **3 ways** to add more employees to your system:

## Method 1: Using Admin Panel (Easiest)

1. **Login as Admin:**
   - Go to: `http://127.0.0.1:5000/admin/login`
   - Admin ID: `ADMIN001`
   - Password: `Admin@123`

2. **Navigate to Employees section**
   - Click on "Employees" in the admin dashboard
   - Click "Add New Employee" button

3. **Fill in the details:**
   - Employee ID: `027` (or next number)
   - Name: `John Doe`
   - Email: `john@company.com`
   - Department: Select from dropdown
   - Click "Save"

4. **Default password** will be the Employee ID (e.g., `027`)
   - Employee will be forced to change password on first login

## Method 2: Direct Google Sheets Entry

1. **Open your Google Sheet** "Daily Attendance"

2. **Go to "Employees" worksheet**

3. **Add a new row** with these columns:
```
ID | Employee ID | Employee Name | Email | Password Hash | Department | Join Date | Is Active | Role | Last Login | Password Changed | Force Password Change | Last Activity | Status | Break Start | Break End | Total Break Time | Break Count
```

4. **For a new employee**, add:
   - ID: Next number (check last ID)
   - Employee ID: `027`
   - Employee Name: `John Doe`
   - Email: `john@company.com`
   - Password Hash: **You need to generate this!** (see below)
   - Department: `Engineering`
   - Join Date: `2026-02-28`
   - Is Active: `Yes`
   - Role: `employee`
   - Force Password Change: `Yes`
   - Status: `offline`
   - Total Break Time: `0`
   - Break Count: `0`
   - Leave others blank

5. **To generate password hash**, add this temporary route:

```python
@app.route('/hash/<password>')
def get_hash(password):
    """Generate password hash"""
    hash_value = generate_password_hash(password)
    return f"Password: {password}<br>Hash: {hash_value}"
```

Visit: `http://127.0.0.1:5000/hash/027` (if you want password = employee ID)
Copy the hash and paste in the Password Hash column.

## Method 3: Add a Quick "Add Employee" Form

Add this temporary route to easily add employees:

```python
@app.route('/quick-add-employee', methods=['GET', 'POST'])
def quick_add_employee():
    """Quick way to add employees"""
    if request.method == 'POST':
        try:
            emp_id = request.form.get('emp_id')
            name = request.form.get('name')
            email = request.form.get('email')
            dept = request.form.get('dept')
            
            sheet = get_google_sheet()
            if not sheet:
                return "Database error"
            
            employees_ws = sheet.worksheet("Employees")
            
            # Check if employee exists
            all_employees = employees_ws.get_all_records()
            for emp in all_employees:
                if str(emp.get('Employee ID', '')).strip() == emp_id:
                    return f"Employee ID {emp_id} already exists!"
            
            # Generate password hash (using employee ID as default password)
            password_hash = generate_password_hash(emp_id)
            today = date.today().isoformat()
            next_id = get_next_id(employees_ws)
            
            # Add employee with all columns
            employees_ws.append_row([
                next_id,           # ID
                emp_id,            # Employee ID
                name,              # Employee Name
                email,             # Email
                password_hash,     # Password Hash
                dept,              # Department
                today,             # Join Date
                'Yes',             # Is Active
                'employee',        # Role
                '',                # Last Login
                '',                # Password Changed
                'Yes',             # Force Password Change
                '',                # Last Activity
                'offline',         # Status
                '',                # Break Start
                '',                # Break End
                '0',               # Total Break Time
                '0'                # Break Count
            ])
            
            return f"""
            <h2>✅ Employee Added Successfully!</h2>
            <p><strong>Employee ID:</strong> {emp_id}</p>
            <p><strong>Name:</strong> {name}</p>
            <p><strong>Default Password:</strong> {emp_id}</p>
            <p><a href="/login">Go to Login</a></p>
            """
            
        except Exception as e:
            return f"Error: {str(e)}"
    
    # GET request - show form
    return '''
    <style>
        body { font-family: Arial; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); padding: 50px; }
        .container { max-width: 500px; margin: 0 auto; background: white; padding: 30px; border-radius: 10px; }
        input, select { width: 100%; padding: 10px; margin: 10px 0; border: 1px solid #ddd; border-radius: 5px; }
        button { background: #667eea; color: white; padding: 12px 20px; border: none; border-radius: 5px; cursor: pointer; }
        h2 { color: #333; }
    </style>
    <div class="container">
        <h2>Quick Add Employee</h2>
        <form method="post">
            <input type="text" name="emp_id" placeholder="Employee ID (e.g., 027)" required>
            <input type="text" name="name" placeholder="Full Name" required>
            <input type="email" name="email" placeholder="Email" required>
            <select name="dept" required>
                <option value="">Select Department</option>

                <option value="HR">HR</option>
                <option value="Finance">Finance</option>
                <option value="Operations">Operations</option>
            </select>
            <button type="submit">Add Employee</button>
        </form>
        <p style="margin-top: 20px; color: #666;">Default password will be the Employee ID</p>
    </div>
    '''
```

Visit: `http://127.0.0.1:5000/quick-add-employee`
